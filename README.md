## Método de sorteo

El sorteo se realiza en vivo por Instagram usando **Sortea2**, plataforma verificada de sorteos online. La lista completa de participantes con sus números se publica antes del sorteo para que todo sea transparente.

El sorteo se activa una vez vendidas un mínimo de 210 chances.
